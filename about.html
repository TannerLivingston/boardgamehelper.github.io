Technology Platforms

Board Game Helper is an Android mobile application developed with the following languages and technologies:

  • C#
  • Xamarin
  • SignalR

The Server and database are hosted on Azure. It has an API for handling game profile saving, updating, and loading, as well as an engine for managing concurrent multiplayer games in real-time. It was developed with the following languages and technologies:

  • C#
  • ASP.NET
  • SignalR
  • Google Firebase
  • Microsoft SQL
